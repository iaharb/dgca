# DGCA Design System Rule: Enterprise High-Density Layouts

All UI components for the KWI Passenger Flow App must adhere to the established DGCA Portal design standards.

## 1. Typography
- **Display/Headings**: Use `Outfit` (sans-serif). Bold and SemiBold weights for hierarchy.
- **Body/System**: Use `Inter`. Clear, legible, and optimized for high-density information.
- **Scaling**:
  - H1: 24sp (Bold)
  - H2: 20sp (SemiBold)
  - Body: 14sp/16sp (Regular)
  - Small/Metadata: 12sp (Medium)

## 2. Color Palette (Enterprise Blue)
- **Primary**: Blue-600 (`#2563eb`) - Actions and focus.
- **Primary-Dark**: Blue-900 (`#1e3a8a`) - Headers and branding.
- **Neutral**:
  - Background: Slate-50 (`#f8fafc`)
  - Surface: White (`#ffffff`)
  - Border: Slate-200 (`#e2e8f0`)
  - Text-Primary: Slate-900 (`#0f172a`)
  - Text-Secondary: Slate-600 (`#475569`)

## 3. Layout & Density
- **Density**: High-density layouts. Avoid excessive whitespace. Use 4dp/8dp base grid for padding/margins.
- **Components**:
  - Borders: 1dp solid Slate-200.
  - Corner Radius: 8dp for cards/buttons (modern enterprise feel).
  - Elevating: Use subtle shadows or borders instead of heavy dropshadows.

## 4. UI Patterns
- **Cards**: Minimalist cards with thin borders.
- **Forms**: Label above input, 12dp vertical spacing between fields.
- **Icons**: Use Google Material Symbols (Rounded) for consistency.
